### About Me

I’m a software engineer based in Cape Town, currently building production warehouse operations systems at Takealot.com. My work spans backend APIs, PostgreSQL systems, Oracle WMS integrations, Flutter scanner applications, auth/permissions services, data extraction pipelines, and operational tooling used in real warehouse environments.

Before Takealot, I worked at MOHARA on AI and data-heavy products, including an AI audience-profiling platform, an AI-facilitated group decision platform, and distributed AWS data pipelines for ecommerce analytics.

I’m especially interested in backend/platform engineering, AI systems, data infrastructure, developer tooling, and products where software has a clear operational impact.

- Currently working with: Python, FastAPI, PostgreSQL, Flutter, Dart, Oracle WMS, Docker, BigQuery
- Strong background in: Django, TypeScript, AWS Lambda, S3, SQS, Redshift, QuickSight, Redis, React, Next.js
- AI experience: LLM evaluation, RAG, prompt orchestration, multimodal judge workflows, AI-assisted development
- Learning focus: Rust, systems programming, distributed systems, and deeper backend architecture
- Engineering style: understand the system, write testable code, verify behaviour, own the outcome

### Selected Work

#### Warehouse Operations Systems at Takealot.com

Built production systems across inventory auditing, cycle counting, Oracle WMS integrations, mobile scanner workflows, auth/permissions, and data pipelines.

Highlights include:

- Built a FastAPI/PostgreSQL and Flutter Honeywell-scanner application for guided cycle counting and targeted stock audits.
- Reverse-engineered Oracle WMS RF/PWA behaviour and built a native Flutter RF scanner client that renders Oracle stateful JSON locally.
- Built a shared FastAPI auth and permissions service used across internal warehouse applications.
- Built a Dockerized Oracle WMS extraction pipeline loading scheduled reporting datasets into BigQuery and Looker.

#### Persuadely / Capuchin

AI audience-profiling and behavioural-science platform.

- Implemented survey ingestion, statistical workbook generation, and customer segmentation using latent class analysis.
- Built AI chat over structured survey data and behavioural-science documents.
- Implemented a multimodal judge/evaluation system that scores competing LLM responses and regenerates weak answers.

#### Chord

AI-facilitated group decision-making platform.

- Built real-time collaborative discussion flows using FastAPI, WebSockets, Celery, and Next.js.
- Helped deliver an MVP demoed successfully to 80 concurrent live users.
- Integrated Anthropic Claude/Opus to facilitate group discussions and consensus-building.

#### Go-Places

Distributed ecommerce analytics platform.

- Built AWS data pipelines using Lambda, S3, SQS, Redshift, EventBridge, CloudWatch, QuickSight, and IAM.
- Processed roughly 2,000 records daily.
- Improved pipeline throughput and reliability through batching, concurrency tuning, and API error handling.

#### DaySpend

iOS budgeting app.

- Built a personal finance app with SwiftUI, StoreKit 2, WidgetKit, and AI-assisted development workflows.
- Used systematic test-driven agent workflows to drive implementation through tests and verification.
